# Tranvideo Wiki

欢迎来到 Tranvideo 项目的 Wiki 文档中心!

## 📖 关于 Tranvideo

Tranvideo 是一个基于 **Whisper Large V3** 和 **Ollama** 的开源视频翻译平台,支持自动语音识别、多语言翻译和字幕生成。

## 🎯 核心特性

| 特性 | 说明 |
|------|------|
| 🎯 智能显存管理 | 先进的显存轮询技术,8GB GPU即可运行 |
| 🌍 多语言支持 | 支持多种语言的识别和翻译 |
| ⚡ 高效处理 | 优化的处理流程,快速生成字幕 |
| 🔧 灵活部署 | Docker/源码/API多种部署方式 |
| 📦 批量处理 | 支持多文件并行处理 |
| 🎬 视频合成 | 自动将字幕嵌入视频 |
