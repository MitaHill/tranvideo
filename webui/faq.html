基础使用

如何获取和使用邀请码？ +

邀请码由系统管理员分发。获得邀请码后：

  1. 在主页输入邀请码
  2. 点击"验证"按钮
  3. 验证成功后即可使用所有功能

邀请码有时间限制，请及时使用。

支持哪些视频文件格式？ +

系统支持常见的视频格式：

  • MP4、MOV、AVI
  • MKV、FLV、WMV
  • WebM、3GP

建议使用MP4格式以获得最佳兼容性。

视频文件大小有限制吗？ +

文件大小限制取决于您的邀请码时长额度：

  • 单个文件最大2GB
  • 批量处理总计不超过5GB
  • 处理时间不能超过邀请码剩余时长

处理相关

字幕识别准确性如何？ +

字幕准确性取决于多个因素：

  • 音频质量：清晰的音频效果更好
  • 语言：中文和英文支持最佳
  • 背景噪音：噪音少准确性更高
  • 语速：正常语速识别效果最佳

一般情况下准确率可达85-95%。

处理需要多长时间？ +

处理时间因文件而异：

  • 字幕生成：约为视频时长的0.3-0.5倍
  • 视频合成：约为视频时长的0.8-1.2倍
  • 队列等待：高峰期可能需要排队

可在状态查询页面查看实时进度。

批量处理有什么限制？ +

批量处理限制：

  • 最多同时处理20个文件
  • 总文件大小不超过5GB
  • 所有文件采用相同处理模式
  • 按顺序依次处理，非并行

技术问题

任务处理失败怎么办？ +

任务失败的常见原因和解决方法：

  • 文件损坏：重新上传原始文件
  • 格式不支持：转换为MP4格式
  • 音频质量差：提供更清晰的音频
  • 系统繁忙：稍后重试

如问题持续，请联系管理员。

处理完成的文件何时过期？ +

文件保存策略：

  • 处理完成后保存24小时
  • 下载后保存3小时
  • 超时自动删除
  • 建议及时下载并备份
如何通过API接口使用系统？ +

API使用指南：

  1. 查看API文档了解接口详情
  2. 使用有效邀请码调用接口
  3. 通过轮询查询任务状态
  4. 任务完成后下载结果文件

所有API都需要有效的邀请码。

故障排除

文件上传失败怎么办？ +

上传失败排查步骤：

  1. 检查网络连接是否稳定
  2. 确认文件大小不超过限制
  3. 验证文件格式是否支持
  4. 清除浏览器缓存重试
  5. 尝试更换浏览器
页面无响应或加载缓慢？ +

性能优化建议：

  • 使用Chrome或Firefox等现代浏览器
  • 关闭不必要的浏览器标签页
  • 检查网络状况
  • 清除浏览器缓存和Cookie
  • 刷新页面重试
邀请码时间不足或已过期？ +

时间管理建议：

  • 处理前检查剩余时长
  • 优先处理较短的视频
  • 避免同时处理多个大文件
  • 联系管理员续期或获取新邀请码
